介绍 ​

登陆组件，兼容 vue2,vue3

使用方式 ​

  • 安装依赖
pnpm install @xxx/xsy-login
  • 登录页初始化
import XsyLogin from 'xsy-login'
import '@xxx/xsy-login/dist/style.css'
  • 开发示例
vue
<template>
  <XsyLogin systemId="1" @msg="loginMsg" />
</template>
<script setup lang="ts">
import XsyLogin from "xsy-login";
import "@xxx/xsy-login/dist/style.css";

const loginMsg = (msg) => {
  ElMessage(msg);
};
</script>

  • 配置属性

    以下是带有是否必填信息的登录组件属性配置选项：

属性名类型描述是否必填默认值
baseUrlstringAPI 请求的基础 URL。是
systemIdstring标识系统或应用程序的 ID。是
clearableboolean是否允许清空输入字段。否false
showPasswordboolean控制密码字段的可见性。否false
countnumber倒计时时间，例如用于 OTP 验证。否60
isForgetPasswordboolean是否显示“忘记密码”选项。否false
isVerificationCodeboolean是否使用验证码登录。否false
isRetrievePasswordboolean是否显示“找回密码”选项。否false
verificationCodeLengthnumber验证码长度。否6
isBuiltInMsgboolean是否使用内置消息。否false
isEncryptboolean是否对敏感数据进行加密。否true
isUserSelfInfoboolean是否在登录时获取用户信息。否false
loginPlatformstring登录平台(PC-WEB：1，APP：2， PC-客户端：3 ，H5：4， 小程序：5)）否"1"
versionstring用于测试的版本标识，可与后端集成以便调试，例如生成表格树结构否
defaultTypenumber登录方式用于 tab 切换多种登录这种情况 (1 是帐号密码登录 2 是验证码登录 3.扫码登录)否1
qrCodeSizenumber二维码显示大小否172
defaultTextstring登陆默认提示文本 placeholder否请输入账号/手机号码

  • 配置方法

    以下是带有是否必填信息的登录组件方法配置选项：

方法名参数类型描述
msg({ message: string; type: string }): void;`不使用内置弹窗提示时使用,显示消息的方法，参数为消息对象。
backLogin(): void;返回登录页面的方法。
forgetPassword(): void;跳转到忘记密码页面的方法。
loginSuccess( val: any): void;登录成功时触发的方法，参数为登录成功后的数据。
loginTypeChange( val: number): void;点击短信登录或者账号登录获取对应的登录方式类型。
